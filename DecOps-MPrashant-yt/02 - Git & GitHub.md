- Initialize git in folder: `git init`
- to check current status: `git status`
- to add file to staging area: `git add file_name`
- to add all files to staging area: `git add *` or `git add .`
![[Pasted image 20250928125515.png]]
Suppose 5-6 friends wants to capture a group photo, so bringing them all on stage means `git add` (staging area) and capturing the photo whoever on the stage is `git commmit`
![[Pasted image 20250928130136.png]]

![[Pasted image 20250928125654.png]]

- to commit: `git commit -m "message to commit"`
- to check logs: `git logs`
- to see a file from old versions/commits: `git show id:file_name`
- to see all files of older commits: `git checkout id --*` or `git checkout id --file_name` (* means all)
- get back from older commit to present: `git checkout current_branch_name`
Note: a single branch have many commits.
